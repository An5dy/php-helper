### 安装
```shell
composer require kakuilan/php-helper
```

### 测试
```sh
php-helper]$ phpunit --bootstrap=tests/bootstrap.php ./tests/

# 或者
cd tests
tests]$ phpunit
```

### 静态检查
```sh
phpstan analyze src
```
